# MioMIP1-Ocean-Intercomparison
This repository contains the Python script that reproduces the figures from [Ocean Meridional Overturning Circulation during the Early and Middle Miocene](https://doi.org/10.1029/2024PA005055). The dataset is available to download at: [Bolin Centre Database](https://bolin.su.se/data/naik-2025-miomip1-moc-1). </br>
Paper DOI: https://doi.org/10.1029/2024PA005055 </br>
Dataset DOI: https://bolin.su.se/data/naik-2025-miomip1-moc-1
